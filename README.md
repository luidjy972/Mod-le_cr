# Modèle de Compte-Rendu — Porte Ouverte

Ce dépôt fournit un **modèle prêt à l'emploi** de compte-rendu pour les événements de type **Journée Portes Ouvertes**.

## Contenu

| Fichier | Description |
|---------|-------------|
| [`compte_rendu_porte_ouverte.md`](compte_rendu_porte_ouverte.md) | Modèle complet de compte-rendu en Markdown |

## Sections du modèle

Le modèle couvre l'ensemble des points nécessaires à un bilan complet :

- **Informations générales** — nom de l'événement, établissement, responsable, date
- **Déroulement** — jours et horaires d'ouverture
- **Fréquentation** — nombre de visiteurs par jour, total et répartition par catégorie
- **Résultats obtenus** — indicateurs quantitatifs (objectifs vs réalisés) et qualitatifs
- **Points positifs** — ce qui a bien fonctionné ✅
- **Points négatifs / axes d'amélioration** — difficultés rencontrées et actions correctives ⚠️
- **Retours des visiteurs** — tableau de satisfaction et commentaires
- **Bilan et recommandations** — synthèse et préconisations pour les prochaines éditions
- **Annexes** — photos, questionnaires, liste des intervenants, etc.

## Utilisation

1. Téléchargez ou copiez le fichier [`compte_rendu_porte_ouverte.md`](compte_rendu_porte_ouverte.md).
2. Remplissez chaque section avec les informations de votre événement.
3. Supprimez ou adaptez les sections optionnelles selon vos besoins.
4. Exportez en PDF ou partagez directement le fichier Markdown.
